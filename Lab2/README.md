# Лабораторна робота №2: Наука про дані: підготовчий етап

## Опис завдання
Це лабораторна робота спрямована на опанування інструментів Python (Pandas, Jupyter, venv) для автоматизованого збору, очищення, статистичного аналізу та передобробки структурованих наборів даних.

* **Частина 1:** Охоплює автоматизацію процесу збору, очищення та структурованої обробки геопросторових даних про індекс VHI з веб-ресурсів за допомогою `pandas` та `urllib`. 
* **Частина 2:** Друга частина лабораторної роботи зосереджена на глибокому аналізі та підготовці великого масиву даних про споживання електроенергії, включаючи складну фільтрацію за критеріями, профілювання часу виконання коду через модуль `timeit`, стандартизація/нормалізація, One Hot coding.

## Вимоги до системи
* **Операційна система:** Windows, macOS, Linux
* **Версія інтерпретатора:** Python 3.9+
* **Залежності:** Необхідні бібліотеки (`pandas`, `jupyter` і тд.) прописані в файлі `requirements.txt`.

## Інструкція із запуску
1. Відкрити термінал у теці `Lab2`.
2. Створити та активувати віртуальне середовище:
   `python -m venv venv`
   `venv\Scripts\activate`
3. Встановити усі необхідні залежності командою:
   `pip install -r requirements.txt`
4. Для успішного виконання "Частина 2" потрібно завантажити файл `household_power_consumption.txt` з репозиторію UCI та покласти в теку `Lab2`.
5. Запустіть Jupyter Notebook в терміналі:
   `jupyter notebook`
6. Відкрити файли `Part1.ipynb` та `Part2.ipynb` і послідовно виконайти комірки.
