# Лабораторна робота №3. Візуалізація даних

## Опис завдання
Обрати унікальний датасет, здійснити Data Cleaning. Навчитися візуалізувати дані за допомогою графіків залежності, гістограм, та складних багатовимірних графіків зі статті Medium.

## Датасет
Для виконання роботи використано **Hepatitis** (з репозиторію машинного навчання UCI).
- **Тип даних:** Багатовимірні (Multivariate).
- **Атрибути:** Категоріальні, цілі та дійсні числа.
- **Data Cleaning:** Виявлено та видалено рядки з відсутніми змінними, для аналізу пацієнтів зі всією інформацією та отримання точних даних.

## Візуалізації (8 графіків)
1. **Scatterplot + Trendline:** Аналіз зв'язку між ключовими біохімічними показниками з відображенням щільності даних (зарубок) безпосередньо на осях координат.
2. **Histogram (5 Bins):** Розподіл пацієнтів за віком у 5 чітких діапазонах з накладанням лінії медіани та виведенням таблиці деталей.
3. **Multidimensional Pairplot:** Попарна матриця розсіювання ключових клінічних характеристик з урахуванням факту виживання (засновано на прикладах зі статей Medium).
4. **4D Bubble Chart:** Одночасна візуалізація чотирьох параметрів (вік, білірубін, альбумін та виживання) за допомогою координат, кольору та розміру точок.
5. **Grouped Violin Plot:** Аналіз щільності розподілу рівня альбуміну за статтю, де кожна фігура розділена навпіл за класом виживання.
6. **Correlation Heatmap:** Теплова матриця кореляції числових медичних атрибутів (відображено лише нижній трикутник для чистоти сприйняття).
7. **Barplot with Annotations:** Горизонтальна стовпчаста діаграма середнього рівня протромбіну в розрізі вікових груп із точними текстовими підписами значень.
8. **Boxplot + Stripplot:** Аналіз розподілу рівня альбуміну залежно від виживання з накладанням точок кожного окремого пацієнта для пошуку викидів.

## Вимоги до системи
* **Операційна система:** Windows, macOS, Linux
* **Версія інтерпретатора:** Python 3.9+
* **Залежності:** Необхідні бібліотеки (`pandas`, `jupyter` і тд.) прописані в файлі `requirements.txt`.

## Інструкція із запуску
1. Створіть та активуйте віртуальне середовище: `python -m venv venv`
2. Встановіть необхідні залежності: `pip install -r requirements.txt`
3. Запустіть Jupyter Notebook: `jupyter notebook`
4. Відкрийте файл з кодом лабораторної роботи та виконайте комірки по черзі.
